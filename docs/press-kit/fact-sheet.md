## Privacy Architecture

Five inviolable rules enforced at the architectural level:

1. **Zero Network in AI Core** — The reasoning process has no networking capability
2. **Gateway Only** — All external communication through a single auditable process
3. **Action Signing** — Every action cryptographically signed and logged before execution
4. **No Telemetry** — Zero analytics, tracking, or crash reporting of any kind
5. **Local Only Data** — All data stored exclusively on the user's device
