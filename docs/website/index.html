Privacy Architecture

5 inviolable rules. Enforced in code.

These aren't guidelines. They're architectural constraints verified by automated audits and confirmed by the source code.

Rule 1

Zero Network in AI Core

The AI Core has no network libraries, no HTTP, no WebSocket, no DNS. Enforced at import level, build level, and OS sandbox level.

Rule 2

Gateway Only

All external communication flows through a single auditable Gateway. Only typed, schema-validated requests to your personal allowlist.

Rule 3

Signed Audit Trail

Every action cryptographically signed and logged before execution. Tamper-evident chain. Review every action Semblance has ever taken.

Rule 4

No Telemetry

Zero analytics. Zero crash reporting. Zero tracking. Not opt-in. Not opt-out. The architecture makes telemetry impossible.

Rule 5

Local Only Data

All data on your device. No cloud sync. No cloud backup. If the device is off, the data is inaccessible. That is the point.
